🔍 OpenCode TUI 的做法

排队：不做本地队列！忙碌时仍可发送，后续消息显示 QUEUED 徽章（纯视觉）。另有独立的 Stash 功能（类似草稿箱）。
子 Agent：每个子 Agent 创建独立 Session。父会话中用一行摘要显示（│ AgentTask — 描述 ↳ 当前工具），点击可跳转到子 Session 查看完整内容。
API：prompt() 是 fire-and-forget（不 await），所有状态通过 SSE 事件推送。
